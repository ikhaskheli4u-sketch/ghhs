## 📌 About Project

This is a modern **School Web Application (ERP System)** developed for **Government Boys High School Sindhri**.
The project helps manage student data, school activities, and administrative tasks efficiently.

یہ ایک جدید اسکول ویب ایپ ہے جو طلبہ، اساتذہ اور اسکول مینجمنٹ کے لیے بنائی گئی ہے۔

---

## 🚀 Features

* 👨‍🎓 Student Management (Add / Edit / Delete)
* 👩‍🏫 Teacher Management
* 🔐 Login System (Admin / Teacher / Student)
* 📊 Dashboard Overview
* 📅 Attendance System
* 📝 Result Management
* 🌐 Multi-language Support (English / Urdu / Sindhi)
* ☁️ Supabase Database Integration
* ⚡ Fast UI (Vite + React + Tailwind CSS)

* ⚛️ React.js

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/your-repo-name.git
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Setup Environment Variables

Create `.env` file and add:

```env
VITE_SUPABASE_URL=your_url
VITE_SUPABASE_ANON_KEY=your_key
```

### 4️⃣ Run Project

```bash
npm run dev
```
